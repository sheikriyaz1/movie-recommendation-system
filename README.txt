# 🎬 AI Movie Recommendation System

This project is an AI-based movie recommendation system built using Python and Streamlit.

## 🚀 Features
- Suggests similar movies using collaborative filtering
- Built with MovieLens dataset
- Interactive UI using Streamlit

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Streamlit

## ▶️ How to Run
```bash
streamlit run app.py
